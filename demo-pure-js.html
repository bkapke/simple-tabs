Simple-Tabs (No jQuery Required)

Simple tabs is designed to be a quick and easy solution for tabbed content on a web page. It is compact, intuitive and light weight. Best of all it can be initialized in one line of code.

Simple-Tabs is dependent on jQuery and a small css and js file.

We are not using the href="#some-hash" convention so the library is not likely to interfere with site navigation in single page apps
